AirPlay Server
===
Feature
----
* Mirror
* Video Play
* Picture Play
* Music Play
* Multi Clients

Support Airplay Client System
-----
* iOS6/iOS7/iOS8/iOS9/iOS10/iOS11

Support Server System
------
* Linux
* Android 4.0+
* Windows

Support Server Hardware
-----
* ARM
* X86_32/X86_64
* MIPS
